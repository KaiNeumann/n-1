## 🚀 Usage Examples

### Basic Usage
```python
from core import get_biomarker

# Get biomarker
crp = get_biomarker("CRP")

# Access ranges
print(crp.ranges["mg/l"])

# Get interpretation
print(crp.interpretation.high)
```

### Search Functionality
```python
from core import search_biomarkers

# Search partial name
results = search_biomarkers("troponin")

# Search by synonym
hba1c = get_biomarker("V-HA1C")  # German lab ID
```

### Clinical Decision Support
```python
# Check if value is in range
biomarker = get_biomarker("Glucose")
for unit, ranges in biomarker.ranges.items():
    for range_obj in ranges:
        if range_obj.min_value <= value <= range_obj.max_value:
            print(f"In range: {range_obj.label}")
```
